Electrophysiological Feature Extraction Toolbox (ElecFeX)

ElecFeX is an open-source MATLAB toolbox for efficient feature extraction from neural recordings. It offers an intuitive GUI, customizable measurements, batch processing, and formatted output.

MATLAB Neural Recording Open Source

Key Features:

  • Intuitive graphical user interface
  • Customizable electrophysiological measurements
  • Batch processing for large datasets
  • Formatted output for further analysis
  • Extensive documentation and examples
Publication: Cell Reports Methods, 2024
GitHub Repository MATLAB Central

Parametric Characterization of Calcium Signals

This MATLAB-based tool automates the analysis of calcium signals by measuring key features such as onset, amplitude, and decay time while distinguishing oscillatory behavior from stochastic noise.

MATLAB Calcium Imaging Signal Processing

Key Features:

  • Automated detection of calcium transients
  • Quantitative parameterization of signal features
  • Statistical analysis of calcium dynamics
  • Modeling of stochastic calcium signals
  • Support for various experimental protocols
Publication: Journal of Theoretical Biology, 2016
